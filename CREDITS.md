# Crédits - High Noon

Merci aux créateurs dont les ressources ont rendu ce projet possible.

## 🎨 Visuels et 3D

Tout le visuel est 100% généré par le code à l'aide de **Three.js**. Aucun modèle 3D externe ou texture importée n'est utilisé.
